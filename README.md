# 🎓 Student Management System

A secure Student Management System built using Java Servlets, JDBC, PostgreSQL, and Bootstrap.

## 🚀 Features

- User Login & Signup
- Add Student
- Update Student
- Delete Student
- Search Student
- Pagination
- Dashboard

## 🛠️ Technologies Used

- Java (Servlet & JSP)
- JDBC
- PostgreSQL
- Apache Tomcat
- Maven
- Bootstrap

## ▶️ How to Run

1. Clone the repository
2. Import into Eclipse as Maven Project
3. Configure PostgreSQL database
4. Run on Apache Tomcat Server
5. Open browser:
   http://localhost:8081/student-management-system
